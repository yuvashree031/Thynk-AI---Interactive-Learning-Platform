# Thynk AI – Interactive Learning Platform

## Overview

Thynk AI – Interactive Learning Platform is a web-based learning application developed using Spring Boot and Java.  
The platform focuses on improving the learning experience through interactive AI-based chat, quizzes, and personalized dashboards.

Students can interact with an AI chatbot, upload study materials, take quizzes based on their preferences, and track their performance.  
The application is designed to be simple, user-friendly, and effective for learning.

---

## Application Endpoints

- **/main**  
  The main landing page of the Thynk AI platform.

- **/login**  
  Allows existing users to log in to their accounts.

- **/signup**  
  Enables new users to create an account and access the platform.

- **/dashboard**  
  Displays the user dashboard with access to learning features.

- **/chat**  
  Interactive AI chatbot page for learning support.  
  Users can communicate with the chatbot and upload files to study more efficiently.

- **/quiz**  
  Allows users to take interactive quizzes based on selected topics, difficulty levels, and number of questions.

- **/score**  
  Displays quiz results and performance summary.

- **/settings**  
  Shows the student profile details and allows light and dark theme customization.

## Tools and Technologies Used

- **Spring Boot (Java)**  
  Backend development and REST API handling.

- **Firebase Console**  
  Used as an online database for storing user and application data.

- **AWS S3**  
  Used to store files uploaded by students for chatbot-based learning.

- **AI Integration**  
  AI services integrated using API keys for interactive chatbot functionality.

- **API Integration**  
  Used for connecting external services and managing data communication.

## Conclusion

Thynk AI – Interactive Learning Platform combines backend development, AI integration, and cloud services to create an interactive and efficient learning environment.  
The project focuses on enhancing student engagement through smart learning tools and personalized features.
